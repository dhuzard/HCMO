# Archived HCMO/MAPP 0.0.1 source

These four modules are the exact canonical source modules superseded by HCMO
0.1.0, apart from the archival banner on the first line. They are retained for
auditability and are not listed in `hcmo.yaml`, so they are not merged into the
current distribution.

The `UNKNOWN:` IRIs, the bare `hcm-obs:` object property, and spreadsheet-derived
class names in this archive are historical Chowlk/export artifacts, not current
HCMO terms. Valid HCMO IRIs that changed in 0.1.0 are retained as deprecated,
mapped terms in `ontology/modules/hcm-compat.ttl`.
