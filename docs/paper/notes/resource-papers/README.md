# Comparable ontology resource papers

Goal: collect examples of ontology/resource papers useful for writing the HCMO resource paper.

| Paper | Venue | Source | Why useful for HCMO | Things to compare with HCMO |
|---|---|---|---|---|
| Odeuropa Data Model | ESWC 2022 | https://2022.eswc-conferences.org/wp-content/uploads/2022/05/paper_136_Lisena_et_al.pdf | Ontology/resource paper with competency questions and SPARQL examples | Resource metadata, CQ table, availability section |
| FidMark | ESWC 2024 | https://2024.eswc-conferences.org/wp-content/uploads/2024/04/146640569.pdf | Recent ontology resource paper with clear resource metadata | Metadata block, figures, evaluation structure |
| POSO: A Generic Positioning System Ontology | ISWC 2022 | https://iswc2022.semanticweb.org/index.php/accepted-papers/ | Ontology using SOSA/SSN for technical observation systems | Sensor/observation/result modelling |
| Procedural Knowledge Ontology | ESWC 2025 | https://zenodo.org/records/15920945 | Ontology built from requirements and use cases | Requirements-to-model structure |
| Purchase-To-Pay Ontology | Resource track | https://openreview.net/forum?id=50N86aYZ3n | Shows competency questions and SPARQL over examples | Query-based evaluation |
