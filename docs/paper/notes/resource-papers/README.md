# Comparable ontology resource papers

Goal: collect examples of ontology/resource papers useful for writing the HCMO resource paper.

| Paper | Venue | Why useful for HCMO | Things to compare with HCMO |
|---|---|---|---|
| Odeuropa Data Model | ESWC 2022 | Ontology/resource paper with competency questions and SPARQL examples | Resource metadata, CQ table, availability section |
| FidMark | ESWC 2024 | Recent ontology resource paper with clear resource metadata | Metadata block, figures, evaluation structure |
| POSO | ISWC 2022 | Ontology using SOSA/SSN for technical observation systems | Sensor/observation/result modelling |
| Procedural Knowledge Ontology | ESWC 2025 | Ontology built from requirements and use cases | Requirements-to-model structure |
| Purchase-To-Pay Ontology | Resource track | Shows competency questions and SPARQL over examples | Query-based evaluation |
